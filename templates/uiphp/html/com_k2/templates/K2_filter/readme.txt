To add new theme for K2 Filter, please make sure you follow our instructions below.
1. Extract your downloaded K2 Filter theme folder
2. Copy the exacted K2 Filter theme folder to: your_site_root_folder/templates/your_template/html/com_k2/
3. Access your site admin panel, then go to: Components >> JA K2 Filter, click on button: Options then select tab: Search Result Item Layout. 
In the tab, you will see the field: Select a template. Now select the added theme.
4. Done
